Kitchen Sink

Functional demos of each available component in UIKit.

All presentational styles are limited to this page, the React components do not come bundled with CSS.
UIButton UICheckbox UICheckboxGroup UIDialog UIImage UIFittedText UIList UIModal UINotification UIPopover UIProgress UIRadio UITypeahead

UIButtonA clickable control with "pressed" state support.

UICheckboxAn accessible checkbox with indeterminate state support.

UICheckboxGroupA controller view for managing the aggregate state of multiple, related checkboxes.

UIDialogA non-blocking, focus-stealing container.

UIImageAn image block with placeholder support for loading and fallback scenarios.

UIFittedTextFit given text inside a parent container, obeying implict and explicit constraints.

UIListA generic list view, supporting unstyled, bulleted and numbered output.

UIModalA blocking, focus-stealing container.

UINotificationTrigger native toasts in supporting browsers.

UIPopoverA non-blocking container positioned to a specific anchor element.

UIProgressAn unopinionated progress implementation that allows for a variety of shapes and effects.

UIRadioAn accessible radio form control.

UITableA high-performance, infinite table view.

UITypeaheadIntelligently recommend entities via customizable, fuzzy recognition.
